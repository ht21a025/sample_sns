# 配布用リポジトリ

## **このリポジトリへのコミット、プッシュなどの編集はしないで下さい。**

---

## フロントエンド

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
